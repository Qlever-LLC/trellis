# Trellis Runtime Service

The core Trellis platform service. Hosts the control-plane runtime, `/auth/*` HTTP endpoints, NATS auth callout, and the `trellis.core@v1` and `trellis.auth@v1` contracts.

Currently implemented in JavaScript/Deno.

See [ADR: Auth](../../../design/adr-trellis-auth.md) and [ADR: Contracts & Catalog](../../../design/adr-trellis-contracts-catalog.md).
