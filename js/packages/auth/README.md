# @qlever-llc/trellis-auth

Authentication helpers for Trellis browser and service clients.

Includes session key management, request signing, browser login/callback helpers, and shared auth schemas.

Pairs with `@qlever-llc/trellis-trellis`, `@qlever-llc/trellis-svelte`, and the generated SDK packages. See [ADR: Auth](../../../design/adr-trellis-auth.md).
