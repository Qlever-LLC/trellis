# Trellis Auth Helpers

Authentication helpers for Trellis browser, service, admin, and device-activation clients.

Includes session key management, request signing, browser login/callback helpers, device activation helpers, and shared auth schemas.

Pairs with `@qlever-llc/trellis`, especially the `@qlever-llc/trellis/auth` subpath, plus `@qlever-llc/trellis-svelte` and the generated SDK packages. See [Trellis Auth Design](../../../design/auth/trellis-auth.md).
